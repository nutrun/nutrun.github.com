rubyworks-ec2 0.1.1

As of version 0.1.1, rubyworks-ec2 encrypts S3 MySQL back ups using the ezcrypto library's AES 128 CBC block cipher. The key used for the encryption transformation is the AWS SECRET ACCESS KEY.
